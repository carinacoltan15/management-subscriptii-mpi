# Management Subscripții MPI

Proiect pentru gestionarea subscripțiilor digitale.

## 🚀 Tehnologii folosite
* **Java 21** cu **Spring Boot 4**
* **PostgreSQL 15** (Rulând în Docker)
* **Hibernate** (ORM pentru persistența datelor)

## 🛠️ Instrucțiuni de pornire
1. **Baza de date:** Intră în folderul `docker` și rulează:
   `docker-compose up -d`
2. **Backend:** Deschide proiectul în VS Code și pornește aplicația din `SubscriptiiApiApplication.java`.
3. **API:** Accesează `http://localhost:8080/api/subscriptions`.
